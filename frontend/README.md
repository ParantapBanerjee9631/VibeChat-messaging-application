# VibeChat Frontend

This is the React frontend for the VibeChat application, bootstrapped with Vite.

## Setup and Scripts

This project uses Vite, React, Zustand, and Socket.IO-client.

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the app for production.
- `npm run lint`: Runs ESLint for code formatting and error checking.
